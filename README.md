# Periodic Table Palace

Static GitHub Pages app for learning the periodic table with a memory-palace workflow.

## Features
- Editable 00-99 number-image system
- Editable vivid images for all 118 elements
- Full 118-locus memory palace
- 7-day intensive training plan
- Local save via `localStorage`
- Firestore scaffolding for future cloud sync
- Version footer fed from `data/version.json`
- GitHub Action that bumps patch version and archives the site on push
