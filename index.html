Offline-first memory trainer

Periodic Table Palace

Build a personal 00-99 number-image system, customise vivid images for all 118 elements, walk a full memory palace, and train with a 7-day intensive plan with daily mini tests.

How to link information

  1. Element image: turn the element into one vivid moving scene.
  2. Atomic number image: convert the number into your 00-99 cue.
  3. Collision: make both images interact absurdly.
  4. Locus: freeze the collision at a specific place in the palace.
  5. Recall loop: say number, symbol, name, image, and location aloud.

Palace training tip
