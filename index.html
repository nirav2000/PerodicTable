Memory palace trainer

Periodic Table Palace

Learn the periodic table with a vivid 00-99 system, exaggerated element scenes, a full 118-locus palace, daily drills, mini tests, and optional Firebase sync for the family.

Design direction

Inspired by the clean spacing, soft panels, and high-clarity hierarchy used across Google Developers ML guides.

iPhone iPad Desktop GitHub Pages

How to link each element

  1. Element scene: turn the element into one vivid moving image.
  2. Number cue: convert the atomic number into your 00-99 image.
  3. Collision: make the two scenes smash into each other.
  4. Locus: freeze that collision in a fixed palace location.
  5. Recall: say number, symbol, name, and location aloud.

Quick start examples
